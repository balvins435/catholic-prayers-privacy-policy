# Privacy Policy

Effective date: 2026-06-08

Catholic Prayers provides Catholic prayers, devotional content, and daily reading links. The app does not require an account and does not ask users to submit names, email addresses, contacts, photos, location, or payment information.

## Information We Process

The app may connect to external services for these limited purposes:

- Daily readings: the app opens official USCCB reading pages from `https://bible.usccb.org` and may request calendar/link information from that site.
- Crash reporting: if configured with a Sentry DSN, the app may send crash reports, error diagnostics, device state, and app version details to Sentry to help diagnose app failures.
- Advertising: if Google Mobile Ads is enabled in a release build, Google may process advertising identifiers and device/ad interaction data according to Google's advertising policies.
- Device integrity check: the app uses device information locally to check whether a device appears rooted or jailbroken. This check is used to show a security warning and is not used to create an account profile.

## Data Sharing

Catholic Prayers does not sell user data. Data processed by third-party services is handled by those providers for the purposes described above.

## Data Security

Network requests use HTTPS. The Android app disables cleartext traffic in the native manifest. Any future certificate pinning changes should be monitored so content requests do not fail when certificates rotate.

## Data Deletion

Catholic Prayers does not create user accounts or store user-submitted personal data on an app-owned server. If crash reporting or advertising services are enabled, requests about data handled by those providers should follow the provider's privacy process.

## Children's Privacy

Catholic Prayers is intended for a general audience and is not designed to collect personal information from children.

## Religious Content Notice

Daily reading links point to USCCB web resources when available. Catholic Prayers is not affiliated with, endorsed by, or sponsored by USCCB, any diocese, parish, or religious organization unless separately stated in an official listing.

## Contact

For privacy questions or data requests, contact Catholic Prayers support using the public support email listed on the app's Google Play Store page.
